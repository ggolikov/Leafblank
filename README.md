## Installation
requires leaflet@1.0.2

browser:

```html
<script src="path/to/leaflet@1.0.2/dist/leaflet.js"></script>
<script src="path/to/Leaflet.plugin.js"></script>
```

browserify:

```
npm install leaflet-plugin
```

```javascript
require('leaflet');
require('leaflet-plugin');
```
